# Getting Started

## Steps to initial setup

### 1. Install nvm 

    $ sudo apt install curl 
    $ curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash 
    $ source ~/.profile  

### 2. Install node

    $ nvm install 15.14.0
    $ nvm use 15.14.0

### 3. Clone Project
    $ git clone https://github.com/rameshmonad/django-frontend

### 4. Ener in project
    $ cd django-frontend

### 5. Install yarn
    $ npm install

### 6. Start Project
    $ 
    $ npm start

### 7. Check react app is working

    http://localhost:3000
